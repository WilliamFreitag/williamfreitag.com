Ping-Pong Java Game

This project is a game called "Ping-Pong"
Ping-Pong is a java program which uses the JavaFX library to respond to keyboard input and create moving graphics.

Click here to go to the github repo!

download the game from the github repo and run Ping-Pong.jar to play

controls:

the "W" and "S" keys move the left paddle up and down
the "I" and "K" keys move the right paddle up and down
